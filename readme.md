# Geoinformatik: Web mapping (26S)

## Klaus Förster, Bernd Öggl, Daniel Schönung (Tutor)

### zur Lehrveranstaltung

* VU3 mit immanentem Prüfungscharakter
* 5 ECTS-AP (entspricht 5*25 Arbeitsstunden, bzw. 9-10 Arbeitsstunden pro Woche)
* Pflichtmodul 5: Geographische Arbeitsmethoden
* Termine: jeweils am Donnerstag von 15:30 - 18:00 Uhr (EDV-Raum)

### Termine

* 05.03. HTML (Hypertext Markup Language) & Arbeitstechniken (Visual Studio Code & GIT)
* 12.03. HTML & Leaflet/Javascript Erstkontakt Hintergrundlayer, Marker, Popup
* 19.03. CSS (Cascading Stylesheets)
* 26.03. Javascript Crashkurs
* **Osterferien**
* 16.04. App Wien: GeoJSON asynchron laden und visualisieren, Layer control, Overlays, Leaflet providers
* 23.04. App Wien: GeoJSON Punkte, Linien und Flächen stylen, Icons verwenden, Leaflet Markercluster
* 30.04. App Wetterstationen: DivIcons und thematische Darstellung, Leaflet RainViewer
* 07.05. App Wettervorhersagen: Map Events, Web-APIs, Popups, Leaflet Velocity
* 21.05. App Bike Trail Tirol: GPX-Dateien, Leaflet Elevation, Leaflet Minimap, Leaflet Fullscreen
* 28.05. CSV to JSON, DOM Manipulation
* 11.06. QGIS GeoJSON Export, TopoJSON, Leaflet Hash, Leaflet Locatecontrol und Tips für die Projektphase
* **Projektphase**
* 25.06.  Präsentationen

### Anforderungen

* 1/3 **Anwesenheit & aktive Mitarbeit**: maximal 2 Einheiten fehlen. Versäumte Inhalte müssen bis zur nächsten Sitzung selbständig nachgeholt werden.
* 1/3 **Workload zwischen den Einheiten**: bis zum jeweiligen Abgabetermin zu erfüllen.
* 1/3 **Gruppenprojekt**: [Web-Applikation mit Präsentation](https://webmapping.github.io/projects)

Alle drei Bestandteile müssen positiv absolviert werden.

### Regeln

* Aufgaben sind eigenständig zu lösen, Tutor zur Unterstützung
* Abgabetermine sind strikt einzuhalten
* durch Fehlen versäumte Inhalte, Arbeitsbeispiele und Workloads sind bis zur nächsten Stunde nachzuholen
